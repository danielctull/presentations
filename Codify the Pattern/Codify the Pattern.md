# Objective-C Conventions in Swift

**Objective-C Convention** | **Swift**
---------------------------|------------------------------------
Initialization             | `convenience` & `designated`
Mutable / Immutable types  | `var` / `let`
nil, null, NSNull, -1      | Optional
Selectors                  | References to functions
